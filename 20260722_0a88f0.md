# 哈利·波特魔法世界观影顺序指南
发布时间：14时30分00秒

## 为何需要按序观看？
《哈利·波特》系列共有八部电影，外加衍生前传《神奇动物在哪里》三部曲。若按上映时间顺序观看，能清晰感受角色成长与魔法世界逐步扩张的脉络；若按故事时间线，则从《神奇动物》系列开始更连贯。

## 故事时间线顺序（推荐）
1. 《神奇动物在哪里》（2016）——讲述纽特在1920年代纽约的遭遇，引出格林德沃的阴谋。
2. 《神奇动物：格林德沃之罪》（2018）——剧情紧接上部，邓布利多与格林德沃的往事浮出水面。
3. 《神奇动物：邓布利多之谜》（2022）——魔法世界大战前夕。
4. 《哈利·波特与魔法石》（2001）——少年哈利初入霍格沃茨。
5. 至《哈利·波特与死亡圣器（下）》（2011）——按上映顺序依次观看即可。

## 观影小贴士
若时间有限，可从《魔法石》直接开始，再回头补前传。两个系列相互独立，但《神奇动物》中的角色与标志性物件会在正传里埋下彩蛋。具体播出平台以官方公布为准。
